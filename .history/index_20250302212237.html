The following are bad UX examples of 3 different websites:

Example 1: Craigslist

Craigslist is a free advertising website where users can post local ads for jobs, housing, items for sale, services, community events, and more.

Some user experience issues with this website:

  • Text-heavy: users will easily feel overwhelmed.
  • Limited use of Colors and Images: users cannot sort things out quickly
  • Navigation Challenge: users are not sure where to start because the filters are not specific.

Example 2: Flipp

Flipp a mobile and web application that displays digital flyers and coupons from retailers to help users save money.

Some user experience issues with this website:

  • Cluttered Design: users can be overhelmed by the layout design (not sure where to start).
  • Hidden Navigation: it creates confusion for user as some key features are hidden in the navigation bar.
  • Unspecific Filter fucntion users are not sure where to start because the filters are not specific.
